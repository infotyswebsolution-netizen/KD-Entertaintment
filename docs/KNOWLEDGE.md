- **Background**: Cinematic collage of movies/TV + semi-transparent dark overlay (60–70%)

## 7. Technical Specifications

### 7.1 Responsive Breakpoints

| Breakpoint | Width | Layout |
|------------|-------|--------|
| Desktop | 1200px+ | Full multi-column |
| Tablet | 768–1199px | 2 columns |
| Mobile | 320–767px | Single column, hamburger nav |

### 7.2 Button Styles

- **Primary CTA**: Navy bg, white text, 14–18px padding, 6–8px radius, optional green dashed border
- **Hover**: Darker blue or slight scale
- **Featured card CTA**: White bg, navy text (inverted)

### 7.3 Spacing Scale

- Small: 10–15px
- Medium: 20–30px
- Large: 40–60px
- Extra large: 80–100px

### 7.4 Performance

- Images: WebP + PNG fallback, lazy loading
- LCP < 2.5s, FID < 100ms, CLS < 0.1

### 7.5 Accessibility

- Contrast ≥ 4.5:1 (WCAG AA)
- Touch targets ≥ 44×44px
- Keyboard navigation
- Focus states visible
