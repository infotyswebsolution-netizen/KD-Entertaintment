- Streaming content thumbnails
- Sports action scenes
- International content from various countries

Visual Style:
- Modern, professional, cinematic aesthetic
- Rich, vibrant colors with good contrast
- Depth of field effect with foreground blur and sharp background
- Professional color grading with warm to cool color balance
- Aspect ratio: 16:9 (1920x1080 minimum)
- Resolution: 4K quality (3840x2160 or higher)

Composition:
- Arrange content in an overlapping, dynamic grid pattern
- Include subtle light rays or lens flare effects
- Add a semi-transparent dark vignette around edges
- Center focal point should be clear for text overlay
- Include subtle motion blur suggesting movement/streaming

Color Palette:
- Navy blue and dark tones as base
- Vibrant accent colors from movie posters
- Gold/warm lighting highlights
- Deep blacks for contrast

Technical Requirements:
- No text or watermarks
- Optimized for web (compressed but high quality)
- Safe for text overlay in center
- Suitable for background image with overlay

**Dimensions:** 1920×1080px min, 3840×2160px preferred | **Format:** WebP/PNG

## Usage Guidelines

1. **Quality**: Request highest resolution available
2. **Consistency**: Keep navy `#003D82` and bright green across all assets
3. **Optimization**: Use web-optimized, compressed files
4. **Transparency**: PNG with transparency for icons and overlays
5. **Naming**: Use consistent names (e.g. `icon-fast-servers.png`, `hero-bg.webp`)
